# 논문 전체 요약

1.2백만 장의 ImageNet LSVRC-2010 이미지를 1000개 클래스로 분류하는 대규모 심층 합성곱 신경망(이후 "AlexNet"으로 불림)을 학습시켜, top-1 오류율 37.5%, top-5 오류율 17.0%로 기존 최고 성능을 큰 폭으로 갱신했다. 네트워크는 6천만 개 파라미터와 65만 개 뉴런, 5개의 합성곱 계층과 3개의 완전연결 계층, 1000-way 소프트맥스로 구성되며, 학습을 가능하게 만든 핵심 요소는 (1) 포화되지 않는 ReLU 뉴런, (2) 2개 GPU에 걸친 효율적 합성곱 구현, (3) 드롭아웃에 의한 과적합 억제였다. ILSVRC-2012 대회에서는 top-5 테스트 오류율 15.3%를 기록해 2위(26.2%)를 압도했으며, 저자들은 깊이 자체가 성능에 결정적이라고 주장한다. (p.1)


### TL;DR

- 1.2백만 장의 ImageNet LSVRC-2010 이미지를 1000개 클래스로 분류하는 대규모 심층 합성곱 신경망(이후 "AlexNet"으로 불림)을 학습시켜, top-1 오류율 37.5%, top-5 오류율 17.0%로 기존 최고 성능을 큰 폭으로 갱신했다.
- 아래 섹션의 수치, 그림, 표 장부를 원문 근거 단위로 확인한다.

## 1 Introduction

**핵심 주장**
- 단순 객체 인식은 작은 데이터셋으로 충분했지만, 현실적 다양성을 가진 객체 인식에는 ImageNet급(1500만 장, 2.2만 클래스) 대규모 데이터가 필요하다.
- 수백만 장 이미지를 학습하려면 학습 용량이 크면서도 사전 지식(prior)으로 보완 가능한 모델이 필요하며, CNN은 깊이/폭 조절이 가능하고 이미지 통계에 대한 강하고 대체로 옳은 가정(정상성, 지역성)을 내장한다.
- 현 세대 GPU(GTX 580 3GB) 2장과 고도로 최적화된 2D 합성곱 구현이 대규모 CNN 학습을 처음으로 실용 범위에 들였다.

**근거·데이터 요약**
- 기여 정리: ILSVRC-2010/2012에서 당시까지 최고 결과, 5 conv + 3 FC 구조, 성능을 개선하는 새 기법들(ReLU, 다중 GPU, LRN, 중첩 풀링), 과적합 방지 기법(데이터 증강, 드롭아웃). 깊이가 중요함 — conv 계층 하나만 제거해도 성능 저하. 학습에 GTX 580 3GB 2장으로 5~6일 소요. (p.1-2)

> [!note] 해석
> 이 서론은 "데이터 규모 + GPU 연산 + 깊은 모델"이라는 이후 10년 딥러닝 스케일링 내러티브의 원형이다. 저자들이 명시한 "결과가 네트워크 크기와 학습 시간에 의해서만 제한된다"는 문장은 사실상 스케일링 법칙 관점의 초기 표명으로 읽을 수 있다.

**원문 페이지**: p.1-2

## 2 The Dataset

**핵심 주장**
- ImageNet은 약 1500만 장, 2.2만 카테고리의 라벨링된 고해상도 이미지 데이터셋이며, ILSVRC 부분집합은 1000개 클래스, 학습 120만 / 검증 5만 / 테스트 15만 장이다.
- 평가는 top-1과 top-5 오류율로 보고한다 (top-5: 정답이 모델이 가장 가능성 높다고 본 5개 라벨에 포함되지 않은 비율).
- 전처리는 최소화: 가변 해상도 이미지를 짧은 변 256으로 리스케일 후 중앙 256×256 크롭, 픽셀별 평균만 차감한 원시 RGB로 학습.

**근거·데이터 요약**
- ILSVRC-2010만 테스트 라벨이 공개되어 있어 주 실험 대상으로 사용; ILSVRC-2012는 대회 결과(테스트 라벨 비공개)로 보고. (p.2)

**원문 페이지**: p.2

## 3 The Architecture

**핵심 주장**
- 8개 학습 계층(5 conv + 3 FC). 중요도 순으로 꼽은 4가지 설계 요소: ReLU, 다중 GPU 학습, Local Response Normalization(LRN), 중첩 풀링.
- ReLU $f(x)=\max(0,x)$ 는 포화 비선형성($\tanh$, sigmoid)보다 경사하강 학습이 수 배 빠르며, 대규모 네트워크 학습을 실질적으로 가능하게 한 요소다.
- 네트워크를 2개 GPU에 분할하되 특정 계층에서만 GPU 간 통신을 허용하는 "기둥(columnar)" 병렬화로 메모리 한계(3GB)를 극복했다.

**근거·데이터 요약**
- **그림 1 (p.3)**: CIFAR-10에서 4계층 CNN의 학습 곡선 비교. x축은 에포크(0~40), y축은 training error rate(0~0.75). ReLU(실선)는 약 6 에포크 만에 학습 오류 25%에 도달하는 반면, $\tanh$(점선)는 같은 25%에 도달하는 데 약 36 에포크가 걸린다 — **6배 빠른 수렴**. 두 곡선 모두 단조 감소하지만 ReLU 곡선이 초반 기울기가 훨씬 가파르다.
- **다중 GPU (3.2, p.3)**: 커널을 GPU당 절반씩 배치, 예컨대 conv3은 conv2의 양쪽 GPU 출력을 모두 받지만 conv4는 같은 GPU의 conv3 출력만 받는 식. 단일 GPU 반쪽 네트워크 대비 top-1/top-5 오류를 각각 1.7%/1.2% 낮추고 학습도 약간 빨랐다.
- **LRN (3.3, p.3-4)**: 수식 보존 —
$$b^{i}_{x,y} = a^{i}_{x,y} \Big/ \left( k + \alpha \sum_{j=\max(0,\,i-n/2)}^{\min(N-1,\,i+n/2)} \left(a^{j}_{x,y}\right)^{2} \right)^{\beta}$$
  하이퍼파라미터 $k=2,\ n=5,\ \alpha=10^{-4},\ \beta=0.75$. 실제 뉴런의 측면 억제(lateral inhibition)에서 영감을 받은 "밝기 정규화". top-1/top-5 오류를 1.4%/1.2% 감소시켰고, CIFAR-10 검증에서 4계층 CNN 오류를 13% → 11%로 낮췄다.
- **중첩 풀링 (3.4, p.4)**: 풀링 간격 $s=2$ < 풀링 창 $z=3$ 의 중첩 구성이 비중첩($s=z=2$) 대비 top-1/top-5를 0.4%/0.3% 낮추고 과적합도 약간 줄였다.
- **그림 2 (p.4-5)**: 전체 아키텍처 다이어그램. 입력 224×224×3 → conv1: 11×11×3 커널 96개, stride 4 → (LRN+풀링) → conv2: 5×5×48 커널 256개 → (LRN+풀링) → conv3: 3×3×256 커널 384개 → conv4: 3×3×192 커널 384개 → conv5: 3×3×192 커널 256개 → FC 4096 → FC 4096 → 1000-way softmax. 다이어그램은 위/아래 두 줄로 그려져 각 GPU가 담당하는 절반을 표현하며, conv3과 FC 계층에서만 두 줄이 교차(통신)한다.

> [!note] 해석
> 그림 2의 입력 크기 224×224는 후속 재구현들에서 zero-padding 산식이 맞지 않아 실제로는 227×227이어야 한다는 것이 정설이 된 유명한 오기(誤記)다. 원문 자체에는 이 언급이 없으므로 재현 시 주의가 필요하다. 또한 LRN은 이후 BatchNorm 계열에 밀려 사실상 폐기된 기법이라는 점에서, 본 절의 4요소 중 현재까지 살아남은 것은 ReLU와 (변형된 형태의) 병렬화 전략이다.

**원문 페이지**: p.2-5

## 4 Reducing Overfitting

**핵심 주장**
- 6천만 파라미터는 ILSVRC 1000클래스 제약(이미지당 10비트 상당)만으로는 심각한 과적합 없이 학습하기에 부족하므로, 데이터 증강과 드롭아웃 두 가지로 과적합과 싸운다.
- 라벨 보존 변환에 의한 데이터 증강은 계산이 사실상 공짜다(GPU 학습 중 CPU에서 파이썬 코드로 생성).
- 드롭아웃은 뉴런 간 복잡한 공적응(co-adaptation)을 깨고 더 견고한 특징을 강제하는, 모델 결합의 효율적 근사다.

**근거·데이터 요약**
- **증강 1 — 크롭/반전 (4.1, p.5)**: 256×256에서 224×224 패치와 좌우 반전을 무작위 추출해 학습셋을 2048배로 확장. 테스트 시 4 모서리 + 중앙의 5개 패치와 각 반전(총 10개)에 대한 softmax 평균으로 예측.
- **증강 2 — RGB 색상 PCA (p.5)**: 학습셋 전체 RGB 픽셀의 PCA를 구해, 각 학습 이미지의 모든 픽셀 $I_{xy}$에 $[\mathbf{p}_1, \mathbf{p}_2, \mathbf{p}_3][\alpha_1\lambda_1, \alpha_2\lambda_2, \alpha_3\lambda_3]^{T}$ ($\alpha_i \sim \mathcal{N}(0, 0.1)$)를 더한다. 조명의 세기/색 변화에 대한 불변성을 포착하며 top-1 오류를 1% 이상 감소.
- **드롭아웃 (4.2, p.5-6)**: 처음 두 FC 계층에 확률 0.5로 적용. 테스트 시에는 모든 뉴런을 사용하되 출력에 0.5를 곱함. 드롭아웃 없이는 상당한 과적합이 발생했으며, 수렴에 필요한 반복 횟수는 약 2배로 증가.

**원문 페이지**: p.5-6

## 5 Details of Learning

**핵심 주장**
- SGD(배치 128, 모멘텀 0.9, weight decay 0.0005)로 학습했으며, 작은 weight decay가 단순 정규화를 넘어 모델의 *학습 오류 자체를* 줄이는 데 중요했다.
- 학습률은 검증 오류가 정체될 때마다 10으로 나누는 휴리스틱을 사용했다.

**근거·데이터 요약**
- 갱신 규칙 수식 보존 —
$$v_{i+1} := 0.9\, v_i - 0.0005\,\varepsilon\, w_i - \varepsilon \left\langle \frac{\partial L}{\partial w}\Big|_{w_i} \right\rangle_{D_i}, \qquad w_{i+1} := w_i + v_{i+1}$$
- 초기화: 가중치는 $\mathcal{N}(0, 0.01)$, conv2/4/5와 FC 은닉층 바이어스는 1(ReLU에 양의 입력을 줘 초기 학습 가속), 나머지는 0.
- 학습률 0.01에서 시작해 종료 전까지 3회 감소; 120만 장 학습셋을 약 90 에포크, GTX 580 3GB 2장으로 5~6일 학습. (p.6)
- **그림 3 (p.6)**: conv1이 학습한 11×11×3 커널 96개의 시각화. 위 48개(GPU 1)는 색상에 무관한 주파수·방향 선택적 필터(가버 유사 에지 필터들), 아래 48개(GPU 2)는 색상 특이적 블롭. 이 분화는 초기화와 무관하게 매 실행 재현됨.

> [!note] 해석
> GPU별 필터 분화(흑백 에지 vs 색상 블롭)는 의도된 설계가 아니라 제한된 연결성(restricted connectivity)이 만든 창발적 특화로, 이후 그룹 합성곱(grouped convolution, ResNeXt 등) 연구의 원형 관찰로 자주 인용된다.

**원문 페이지**: p.6

## 6 Results

**핵심 주장**
- ILSVRC-2010에서 top-1 37.5% / top-5 17.0%로 종전 최고(45.7% / 25.7%)를 대폭 갱신했다.
- ILSVRC-2012에서 CNN 앙상블로 top-5 테스트 오류 15.3%를 달성, 2위(26.2%)와 약 11%p 격차를 냈다.
- 마지막 은닉층(4096차원) 활성 벡터는 의미적 유사성을 인코딩하므로 이미지 검색 등에 쓸 수 있는 특징 표현이다.

**근거·데이터 요약**
- **표 1 (p.7)** — ILSVRC-2010 테스트셋 비교 (모델 / top-1 / top-5):
  | 모델 | Top-1 | Top-5 |
  |---|---|---|
  | Sparse coding (당시 대회 우승) | 47.1% | 28.2% |
  | SIFT + FVs (당시 발표된 최고) | 45.7% | 25.7% |
  | **CNN (본 논문)** | **37.5%** | **17.0%** |
- **표 2 (p.7)** — ILSVRC-2012 검증/테스트 오류율: SIFT+FVs는 top-5 26.2%; 1 CNN은 40.7%/18.2%(val top-1/top-5); 유사 CNN 5개 평균은 38.1%/16.4%, 테스트 16.4%; ImageNet 2011 Fall 릴리스로 사전학습한 1 CNN*은 39.0%/16.6%; 사전학습 CNN 2개를 더한 7 CNN* 앙상블은 36.7%/15.4%, **테스트 top-5 15.3%**.
- ImageNet 2009 가을 버전(10,184 클래스, 890만 장)에서는 top-1 67.4% / top-5 40.9%로 종전 발표치(78.1%/60.9%)를 크게 상회. (p.7)
- **그림 4 좌 (p.8)**: 테스트 이미지 8장과 각각의 top-5 예측 막대그래프(정답 라벨은 빨간 막대). 진드기처럼 구석에 치우친 객체도 인식하고, 표범(leopard) 오답 후보들이 재규어·치타 등 시각적으로 타당한 범주임을 보여줌. 그릴(grille)·체리처럼 의도된 초점이 모호한 경우가 주요 실패 양상.
- **그림 4 우 (p.8)**: 첫 열의 테스트 이미지 5장 각각에 대해, 마지막 은닉층 특징 벡터의 유클리드 거리가 최소인 학습 이미지 6장을 나열. 코끼리·개 등이 포즈/배경이 달라도(픽셀 L2로는 전혀 가깝지 않음) 의미적으로 같은 클래스가 검색됨 — 깊은 표현이 의미 공간을 형성한다는 정성적 증거.

**원문 페이지**: p.7-8

## 7 Discussion

**핵심 주장**
- 크고 깊은 CNN이 순수 지도학습만으로 기록 갱신 성능을 달성할 수 있으며, **깊이가 정말로 중요하다** — 중간 conv 계층 하나만 제거해도 top-1 성능이 약 2% 손실된다.
- 비지도 사전학습 없이 이 결과를 얻었고, 더 큰 네트워크와 더 긴 학습으로 성능이 계속 오를 것으로 기대한다.
- 최종 목표로 시간 구조가 정보를 주는 비디오 시퀀스에 매우 크고 깊은 CNN을 적용하고 싶다고 밝힌다.

**근거·데이터 요약**
- 인간 시각계의 하측두엽 경로(infero-temporal pathway) 대비 네트워크가 여전히 수십 배 작다는 정량 비교로 확장 여지를 주장. (p.8)

> [!note] 해석
> "계층 제거 시 2% 손실"은 단일 절제(ablation) 관찰일 뿐 통제된 깊이-성능 곡선은 아니다. 깊이의 인과적 중요성은 이후 VGG(2014), ResNet(2015)에서야 체계적으로 입증되며, 이 논문의 해당 주장은 방향성 가설에 가깝다.
