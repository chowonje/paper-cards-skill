# Generation Cautions

These cautions come from observed paper-card review failures. Apply them while reading, writing, and reviewing.

## Active Cautions

1. Graph values are easy to overread. If a value is estimated from a curve, cross-check it against text, captions, or tables. If no cross-check exists, write an approximate trend rather than an exact value.
2. Do not invent examples or labels. Reproduce only examples and panel labels actually printed in the paper.
3. Do not collapse large comparison tables so far that the competing models, baselines, or conditions disappear.
4. Keep frontmatter `source` limited to information confirmed inside the PDF.
5. Use physical PDF page numbers. Printed page numbers can be added only as a secondary reference.
6. For long papers, inspect appendices, system cards, supplements, and referenced figure/table pages before declaring coverage complete.
7. Do not mix author claims and reviewer interpretation. Interpretation belongs only in `> [!note] 해석`.
8. Avoid long direct quotation. Public examples should be summaries with page references, not reconstructed paper text.
